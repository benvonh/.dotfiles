# .dotfiles

My dotfiles.

## Dependencies

- neovim (included as app image)
    - `fd-find`
    - `nodejs`
    - `ripgrep`
    - `unzip`
    - `yarn`

- ranger (install)

- zsh (install)

## Install

Run the install script :)

## Uninstall

Replace symbolic links under `~/.config/` and remove `~/.zshrc`. Cached and local files may remain from various apps.
