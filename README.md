# .dotfiles

My dotfiles.

## Dependencies

- neovim (included app image)
    - `fd-find`
    - `nodejs`
    - `ripgrep`
    - `unzip`
    - `yarn`

- ranger (requires install)

- zsh (requires install)
    - `tree`

## Install

Run the install script :)

## Uninstall

Replace symbolic links under `~/.config/` and remove `~/.zshrc`.
Cached and local files may remain from various apps.
